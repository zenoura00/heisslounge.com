## Navigation Flow
1. **Splash Screen** → First load only
2. **Home (/)** → Links Page with social media buttons
3. **Menu (/menu)** → Categories grid with accordion style
4. **Menu Details (/menu/[slug])** → Menu items with prices

## Project Overview
- Next.js 16 with Tailwind CSS
- Elegant restaurant/lounge menu for "Heiss Lounge"
- **Flow**: Splash → Links Page → Menu → Menu Details
- Features: Splash screen, Links page, mobile menu, menu categories with expandable items
- Menu categories: Alcoholic Beverages, Non-Alcoholic Beverages, Sweets & Snacks, Shisha
